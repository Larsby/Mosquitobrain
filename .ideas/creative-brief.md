# Mosquitobrain Creative Brief

## Hook
Mosquitobrain is a compact neural tone transfer plugin that learns the transformation between an input sound and a target output sound, then applies that learned behavior in real time as an effect.

## Vision
The plugin is built for fast experimentation: capture your source and target examples, train a small model quickly, and audition the result immediately inside your DAW.

## Sonic Character
Adaptive, data-driven tone shaping that can range from subtle coloration to strong character transformation, depending on training material.

## Core Experience
- Train and retrain directly from user-provided input and output audio examples.
- Save and recall trained models as presets.
- Blend learned processing with the original signal using a dry/wet control.

## Scope for Phase 1
- Define product concept and key controls for first implementation.
- Keep UX minimal and modern.
- Prioritize reliable train, retrain, and preset recall workflow.
